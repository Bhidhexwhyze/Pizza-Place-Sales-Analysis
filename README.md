# Pizza Place Sales Analysis

This project analyzes one year of pizza sales data to generate business insights including revenue trends, best-selling pizzas, peak sales hours, and performance by category.

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn

Key Insights:
- Peak sales occur during evening hours
- Friday generates the highest revenue
- Large-size pizzas contribute most to revenue
